Online Feedback System

Technologies used - HTML, CSS, Xampp, PHP, MySQL

OS - Windows 8.1(64 bit)

Team - 2 Members

Description - It's a project which provides the feedback system for Students and Lecturers. It is build such a way that an Admin can only add a student or a lecturer to this system. Students can provide feedback, only to respective semester Lecturers. Also, Lecturers can only see the feedback not knowing the respective student. If any query, both Lecturers and Students can contact Admin by droping a message in contact section. Overall Admin has controlling powers of this system.

Online Adoption Agency for Child Adoption from Orphanages

Technologies used - HTML, CSS, Firebase

OS - Windows 8.1(64 bit)

Team - 4 Members

Description - This is a Website which acts a agency between Parents and Orphanages for Child Adoption. In this website, parents who are want information abiut Adoption can know about the Adoption Process, Government guidelines for Adoption, Eligibility Criterias and many more. If they are Interested in Adoption, they can register(if new user) or login(if already a user) and make a Adopion Request by filling a form about their complete details with any preferences for child(only such as age, color, religion, gender, health condition). They can also check their request Status. This Website also provides Counseling Programme, Baby Caretaking Programme details if any parents are interested. Overall this website provides all the necessary information and provides a platform where parents can easily start the Adoption process. All the legal Works are conducted only after the background verification of the parents.
